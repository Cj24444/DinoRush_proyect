# DinoRush

## Descripción

Juego de la categoría "endless runner", inspirado en la pantalla de carga de Google cuando no hay conexión con la red. 

El jugador controla a un personaje que avanza automáticamente y debe esquivar obstáculos para obtener la mayor puntuación posible.

El proyecto tiene como objetivo aplicar buenas prácticas de desarrollo utilizando Git como sistema de control de versiones, documentando la evolución del software mediante commits, ramas, merges y resolución de conflictos.

## Estado del proyecto

En desarrollo.

## Tecnologías utilizadas

* Vue 3
* Vite
* JavaScript
* HTML5
* CSS3

## Funcionalidades planificadas

* Pantalla de inicio.
* Inicio de partida.
* Movimiento automático del personaje.
* Salto.
* Obstáculos generados aleatoriamente.
* Detección de colisiones.
* Sistema de puntuación.
* Pantalla de Game Over.
* Reinicio de la partida.
